Custom Yandex Maps

Простая карта с одной меткой

  • При клике на метку появляется балун в котором содержится заголовок и текст.
new CustomMaps({
    selector: 'demo-1a',
    coords: [55.050432, 60.109599],
    balloonContentHeader: 'Миасс',
    balloonContent: 'Город в золотой долине',
});

Простая карта с одной меткой

  • При клике на метку появляется попап в котором имеется: заголовок, описание, телефоны, почта.
Список объектов
sss 
new CustomMaps({
    selector: 'demo-1b',
    coords: [55.050432, 60.109599],
    popup: true,
    popupName: 'Миасс',
    popupDescription: 'Город в золотой долине',
    popupPhones: {
        label: 'Телефоны:',
        value: ['+79511111111', '+79512222222']
    },
    popupEmails: {
        label: 'Почтовый ящик:',
        value: ['example@mail.ru', 'example@mail.ru']
    }
});
